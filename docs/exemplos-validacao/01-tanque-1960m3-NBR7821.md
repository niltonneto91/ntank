**Fórmula utilizada na planilha (NBR 7821 simplificada):**

```
e = 0,04 × D × (H_efetiva − 0,3) × G + CA
```

Onde:
- `e` em mm
- `D` em **metros**
- `H_efetiva` em **metros** = soma das alturas de **todos os anéis a partir do anel atual até o topo** (i.e., a coluna líquida acima da base do anel atual)
- `G` adimensional
- `CA` em mm

A planilha aplica `max(e_calc, t_min_nominal)` e arredonda para a chapa comercial superior.

## Fórmula NBR 7821 simplificada vs API 650 1-Foot

A planilha usa uma versão simplificada da NBR 7821. A relação com a API 650 1-Foot é:

```
NBR 7821 simplificada:  e = 0,04 × D × (H − 0,3) × G + SC
API 650 1-Foot:         e = 4,9 × D × (H − 0,3) × G / (Sd × E) + CA
```

A constante 0,04 corresponde a `4,9 / (Sd × E)` para o caso típico:
- `Sd = 137 MPa` (A283-C, valor típico para tanques de combustível na época da norma)
- `E = 0,85` (radiografia parcial)
- `4,9 / (137 × 0,85) = 4,9 / 116,45 ≈ 0,04208` ≈ 0,04 ✓

A planilha embute essa simplificação porque é otimizada para aço carbono comum em tanques de combustível convencionais. **No NTANK, vamos implementar:**

1. `nbr7821Simplificada(D, H, G, CA)` — fórmula direta da planilha (case insensitive ao material).
2. `api650OneFoot(D, H, G, CA, Sd, E)` — fórmula geral com material e eficiência explícitos.

Os dois métodos devem **convergir numericamente** quando `Sd = 137`, `E = 0,85`, dentro de ±0,005 mm.
